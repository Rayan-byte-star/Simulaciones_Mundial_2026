# Simulaciones Mundial 2026 ⚽🏆

¡Bienvenido al repositorio de **Simulaciones Mundial 2026**! Este espacio está dedicado al desarrollo, análisis y ejecución de algoritmos de predicción y simulación para las distintas fases de la Copa Mundial de la FIFA 2026. 

Aquí encontrarás modelos y scripts diseñados para proyectar los posibles escenarios del torneo, desde las rondas de eliminación directa hasta la gran final.

---

## 📂 Estructura del Repositorio

El proyecto está organizado en módulos independientes según la fase del campeonato simulada:

*   **`Simulador-16avos_mundial-2026-main/`**: Scripts y lógica para la ronda de dieciseisavos de final.
*   **`Predicción_Octavos_Mundial_2026-main/`**: Modelos de predicción y emparejamientos para los octavos de final.
*   **`Predicción_Cuartos_Mundial_2026-main/`**: Simulaciones avanzadas y proyección de cruces para los cuartos de final.

---

## 🚀 Características del Proyecto

*   **Simulación Dinámica:** Algoritmos adaptables basados en rendimiento histórico, estadísticas de selecciones y variables aleatorias de fútbol.
*   **Modularidad:** Cada fase del torneo cuenta con su propia lógica y datos de entrada, lo que facilita realizar pruebas aisladas.
*   **Análisis de Probabilidades:** Generación de múltiples escenarios para entender qué selecciones tienen más opciones de avanzar en cada cruce.
